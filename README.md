# ⚽ Football Performance Analysis

Projeto de análise de desempenho físico no futebol utilizando Python.

## 📊 Objetivo
Analisar métricas de desempenho para suporte à tomada de decisão em contexto esportivo.

## 🧠 Tecnologias
- Python
- Pandas
- Excel

## 📈 Possíveis análises
- Volume de treino
- Intensidade
- Comparação entre atletas
